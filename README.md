# Store-Management-System
Store Management System
A robust Store Management System built with HTML for the frontend and a backend powered by JavaScript and Python. The system manages product inventory, tracks quantities, updates prices in real-time, and supports secure online payments via QR code. Data is stored and managed using a MySQL database.

Features
Product Quantity Management: Monitor and update product quantities effortlessly to keep inventory accurate.

Real-time Price Updates: Modify product prices instantly to reflect current pricing or discounts.

Online Payment via QR Code: Integrated QR code payment system for fast and secure online transactions.

MySQL Database Integration: Reliable backend storage for products, inventory, and payment data.

Clean and Responsive UI: Built with HTML for an intuitive and accessible user experience.

Technologies Used
Frontend: HTML, CSS, JavaScript

Backend: JavaScript (Node.js or your framework), Python (for specific backend tasks or APIs)

Database: MySQL

Payment Integration: QR code-based online payment gateway
